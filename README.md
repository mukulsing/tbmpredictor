# TBM Outcome Predictor

AI-powered 6-month outcome predictor for Tuberculous Meningitis patients.

- **Model:** Logistic Regression (5-Fold CV)
- **AUC:** 0.863 | **Accuracy:** 80.6%
- **Trained on:** 57 TBM patients

## Deploy on Render

1. Push this folder to a GitHub repository
2. Go to [render.com](https://render.com) → **New** → **Static Site**
3. Connect your GitHub repo
4. Set these settings:
   - **Build Command:** *(leave empty)*
   - **Publish Directory:** `.`
5. Click **Deploy** — live in ~1 minute!
